Instructions to setup and run YOLOv5 on EC2

1. ssh into your EC2 instance
“ssh -i w251.pem ubuntu@12.345.678.910”

2. cd “capstone” (or project folder where you have data)

3. clone YOLO git folder
git clone https://github.com/ultralytics/yolov5

4. cd “yolov5”

5. install binaries as defined in “requirements.txt”
pip3 install -r requirements.txt

6. cd .. (go back to the “capstone” folder)

7. create capstone.yaml file (vi capstone.yaml) with following:
# capstone.yaml
# train and val data as 1) directory: path/images/, 2) file: path/images.txt, or 3) list:
[path1/images/, path2/images/]
train: ../train/images/
val: ../val/images/
# number of classes
nc: 2
# class names
names: [ 'fox', 'snowshoehare' ]

8. above yaml file can be changed to add more classes

9. create folders “train/images”, “train/labels”, “val/images”, “val/labels” under
“capstone” folder

10. copy images in the “train/images” and “val/images” folders (80%-20%) and
corresponding yolo txt files in the “train/labels” and “val/labels” folders

11. cd “yolov5”

12. run training with the following command:
python3 train.py --img 329 --batch 8 --epochs 100 --data
'../capstone.yaml' --cfg ./models/yolov5s.yaml --weights '' --name
yolov5s_cap_results

My version: 
python3 train.py --img [329] --batch [8] --epochs [100] --data
'../w210-capstone-yolov5.yaml' --cfg ./models/yolov5s.yaml --weights '' --name
yolov5s_cap_results

13. “runs” folder should have the results
